* SOBRE O SISTEMA *


O sistema tem o objetivo de Ler e Transmitir arquivos de vídeo de um dispositivo "raiz" para outros que estiverem conectados à mesma rede! O sistema NÃO FORNECE ARQUIVOS NEM PROMOVE A PIRATARIA, é apenas uma solução prática para o consumo de arquivos próprios.


* COMO VER OS VIDEOS *


Para alterar o dirtório dos arquivos, siga este passo a passo: Abra o arquivo "app.py" no diretório onde o projeto está salvo. Procure pela variável PASTA_VIDEOS . Após isso, digite ou cole o caminho dos arquivos que serão mostrados no site.
